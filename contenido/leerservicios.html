Leer servicios desde otros servicios

Usar un servicio desde otro servicio no es tan sencillo debemos injectar el servicio que usaremos

PersonasService.service.ts

import { persona } from './persona.model';
import { LoggingService } from './LoggingService.service';
import { Injectable } from '@angular/core';

@Injectable()
export class PersonasService{
  personas:persona[] = [new persona("Veronica","Hernandez"), new persona("Gisela", "Perez")];
  
  constructor (private LoggingService:LoggingService){}
  onPersonaAgregada(event:persona){
    this.LoggingService.imprimirMensaje("Persona agregada " + event.nombre);
    this.personas.push(event);
  }
}

Debemos importar la clase Injectable y utilizarla arriba de la clase. En el contrstructor declaramos el servicio que vamos a usar y por ultimos lo llamamos usando this
