Property binding entre componentes

Esto sirve para mandar datos entre componente de padre a hijo

app.component.html

<app-persona *ngFor="let personaArray of personas; let i = index"
[persona]="personaArray"
[index]="i"
></app-persona>

Usamos el property binding para pasar el arreglo de persona por la propiedad persona y la varialbe i para pasar el indice

persona.component.ts

import { Component, OnInit, Input } from '@angular/core';
import { persona } from '../persona.model';

@Component({
  selector: 'app-persona',
  templateUrl: './persona.component.html',
  styleUrls: ['./persona.component.css']
})
export class PersonaComponent  {
@Input()persona:persona;
@Input()index:number;
}

Devemos agregar el componente input en el import

persona.component.html

<p>{{index+1}}.- {{persona.nombre}} {{persona.apellido}}</p>
