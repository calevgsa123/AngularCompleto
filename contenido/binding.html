Property binding

Esto es similar al consepto de interpolacion solo que para usar el valor de una variable en una propiedad de un componente HTML devemos poner la propiedad en corchetes y el nombre de la barible entre comillas

App.component.ts

import {Component} from '@angular/core';

@Component({
    selector: 'app-personas',
    templateUrl: './personas.component.html'
})

export class PersonasComponent{
    agregarPersona=true;// Agregamos en la declaraci√≥n de la clase la variable que usamos para relacionar el boroton

}

app.component.html

<div class="container">
    <div class="row">
        <div class="col">
            <button [disabled]="agregarPersona">Agregar persona</button>
        </div>
    </div>
</div>
