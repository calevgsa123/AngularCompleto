Usar el host de firebase

Podemos usar el host de firebase para subir nuestra aplicacion y asi publicarla, si no exede la cuota permitida para un servicio gratis no hay problema pero si tiene devaciadas solicitudes habra que pagar un servicio mas avanzado

Lo primero es instalar las herramientas de firebase

>npm install -g firebase-tools

Despues ejecutamos los comandos sigientes y vamos siguiendo las instrucciones

>firebase login

>firebase init

///Tenemos que compilar el proyecto antes de subirlo
>ng build --prod --aot

//Y modificar el archivo firebase.json con la ruta donde se creo: dist/nombre_proyecto

>firebase deploy
