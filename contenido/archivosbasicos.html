Archivos basicos

A la hora de crear el primer proyecto se crean a su ves varios archivos basicos para que angular funcione, ellos son:

app.component.ts

Este es el archivo TypeScript donde va el codigo de nuesta componente app.component.html

import { Component } from '@angular/core';//Se importa el componente core de angular

@Component({//Decorador con los datos del componente
    selector: 'app-root', //nombre del selector con el que se llamara el control
    templateUrl: './app.component.html',//template html osea la pagina que interactuara con el codigo
    styleUrls: ['./app.component.css']//Estilos de la pagina
})
export class AppComponent {//Se exporta la clase
    title = 'Hola mundo desde Angular';//Propiedad declarada
}

app.component.html

Este es el archivo de tipo html donde se escribe todo lo visual y de donde se llaman los componentes

 <p>{{title}}</p> //Aqui usamos la interpolacion que ponemos entre llaves la propiedad y se impÂ´rime en el navegador

app.module.ts

Este archivo guarda los recursos del proyecto y es donde se configuran y declaran componentes, proveddores, y demas herramientas para trabajar como el proyecto valla avanzando se este archivo ira creciendo

import { BrowserModule } from '@angular/platform-browser';
import { NgModule } from '@angular/core';

import { AppComponent } from './app.component';
import { PersonasComponent } from './personas/personas.component';

@NgModule({
    declarations: [
    AppComponent, PersonasComponent
    ],
    imports: [
    BrowserModule
    ],
    providers: [],
    bootstrap: [AppComponent]
})
export class AppModule { }

index.html

Es el archivo principal donde se va a mostrar todo tanco componentes como demas objetos y configuracion
